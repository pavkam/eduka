Процедура

Процедуры являются вспомогательными алгоритмами которые выполняют последовательность команд в момент вызова.

 

В основном алгоритме могут быть написаны вспомогательные алгоритмы. Вспомогательные алгоритмы объявляются перед словом Начало с помощью ключевого слова Процедура с указанием Имени процедуры.

Вызов процедуры выполняется в теле основного алгоритма с помощью ключевого слова Вызов с указанием Имени процедуры.

Процедура описывается следующим образом:
 Процедура <Имя Процедуры>
   ...
  последовательность команд
   ...
 Конец Процедура

 

Заголовок процедуры содержит:

<Имя Процедуры> - имя процедуры.

В теле процедуры включена последовательность команд, которая будет выполнятся в момент вызова.
