Каталог, используемый по умолчанию

Для быстрого нахождения документов с которыми работаете, есть возможность использовать по умолчанию каталог Microsoft Windows или указание что Eduka+ использует по умолчанию другой рабочий каталог.

 

Microsoft Windows имеет каталог названный My Documents (Мои Документы), который может быть местом для хранения ваших документов. Если работаете с различными документами Eduka+, есть возможность хранения разных по типу документов в подкаталоги, которые находятся в каталоге Мои Eduka+ Файлы из каталога Мои Документы.

 

Впервые когда открываете диалоговые окна Открыть Файл и Сохранить Файл, после чего запускаете приложение Eduka+, каталог Мои Eduka+ Файлы является рабочим каталогом, используемый по умолчанию (или текущим).

 

Имеете возможность заметить, что Eduka+ использует различные каталоги для хранения алгоритмов, задач, тестов, отчетов и html страничек.
