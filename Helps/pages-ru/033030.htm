Создание теста

В меню Инструменты выберите команду Редактор Тестов.
Программа переходит в режим редактирования тестов.

 

Следующее действие создает новый незаполненный тест.

На панели инструментов, выберите команду Создать новый тест.

Из меню:
Выберите команду Создать из меню Файл.

 

Созданный с помощью Eduka+ тест формируется из одной или более задач.

Для добавления задачи в тест, используйте кнопку Пролистать каталог для выбора каталога который содержит задачи.
Eduka+ начнет поиск в глубину выбранного каталога (время поиска зависит от содержимого каталога) и выведет список найденных задач.
Далее используйте кнопку Добавить задачу для добавления выделенной задачи из списка найденных задач в список выбранных задач.
Соответственно используйте кнопку Удалить задачу для удаления выделенной задачи из списка выбранных задач.

 

Совет:
Используйте следующие критерии фильтрации в списке найденных задач.
Длительность - фильтрация по времени отведённому для ркшения задачи
Уровнь - фильтрация по уровню сложности
Исполнитель - фильтрация по типу исполнителя
