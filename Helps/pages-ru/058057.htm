Предметный указатель

 

Панель команд для исполнителя Кенгурёнок

Чтобы вставить в зону редактирования алгоритма простую команду или структуру составной команды, щелкните соответствующую кнопку, названия которых являются зарезервированными словами.
Чтобы вручную управлять Кенгуренком, щелкните кнопки, рисунок которых подсказывает поведение Кенгуренка при выполнении простых команд.

Создать

Щелкните, чтобы создать новый документ.
Эту команду можно выбрать и в меню Файл.

 

Открыть

Щелкните чтобы вывести диалоговое окно Отрыть Файл, для открытия документа.
Эту команду можно выбрать и в меню Файл.
Чтобы открыть один из недавно открывавшихся документов, выберите его имя в списке недавно использованных документов из меню Файл.

Панель Условия задачи

На этой панели отображается условия и ограничения текущей задачи.

 

Поле Автор

Введите Имя и Фамилию автора задачи.

 

Поле Длительность/Уровень

Выберите Время, в минутах, отведенное для решения задачи.
Выберите Уровень сложности задачи.

 

Поле Язык

Выберите Язык, на котором будет указанно название, условия и ограничения задачи.
Можно ввести данные на нескольких языках.

 

Поле Название задачи

Введите Название задачи для выбранного языка.

 

Поле Условия задачи

Введите Условия задачи (начальные/конечные) для выбранного языка.

 

Поле Ограничения задачи

Введите Ограничения задачи (в использовании/не использовании некоторых команд) для выбранного языка.

 

Начальное/ конечное состояние поля

Выберите для редактирования начальное или конечное состояние поля.

 

Список Языки

Выберите из списка язык, на котором будут отображаться название, условия и ограничения задачи.

 

Поле Название задачи

Здесь отображается название текущей задачи для выбранного языка.

 

Поле Автор

Здесь отображается имя и фамилия автора текущей задачи.

 

Поле Исполнитель

Здесь отображается исполнитель текущей задачи.

 

Поле Масштаб поля

Здесь отображается масштаб поля текущей задачи.

 

Поле Длительность

Здесь отображается время, в минутах отведенное для решения текущей задачи.

 

Поле Уровень

Здесь отображается уровень сложности текущей задачи.

 

Поле Условия задачи

Здесь отображается условия текущей задачи для выбранного языка.

 

Поле Ограничения задачи

- фильтрация по уровню сложности
