Места для хранения файлов

Каталог Мои Eduka+ файлы

Когда открываете диалоговые окна Открыть файл и Сохранить файл, каталог Мои Eduka+ файлы появится по умолчанию. Каталог Мои Eduka+ файлы хорошее место для хранения файлов, с которыми работаете, на пример алгоритмы, задачи и тесты, сгруппированные по типу.

 

В каталоге Мои Eduka+ файлы вложены следующие каталоги:

Каталог Исходные алгоритмы
Этот каталог используется для хранения созданных алгоритмов.
В каталоге Исходные алгоритмы вложен каталог Сборник – коллекция примеров готовых алгоритмов, предназначенных для демонстрации программы.

Каталог Задачи
Этот каталог используется для хранения созданных задач.
В каталоге Задачи вложены каталог Примеры – коллекция примеров готовых задач, предназначенных для демонстрации программы и каталог Html – место, отведенное для хранения страничек в формате Html, созданные с помощью Eduka+.

Каталог Тесты
Этот каталог используется для хранения созданных тестов.
В каталоге Тесты вложены каталог Примеры – коллекция примеров готовых тестов, предназначенных для демонстрации программы и каталог Html – место, отведенное для хранения страничек в формате Html, созданные с помощью Eduka+.
