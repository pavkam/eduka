Отладка алгоритма

Пошаговое выполнение алгоритма

Следующее действие реализует пошаговое выполнение текущего алгоритма
В меню Программа выберите команду Следующий Шаг.

 

Визуальная демонстрация пошагового выполнения алгоритма

Следующее действие выделяет другим цветом в редакторе текущий выполненный оператор.
В меню Опции выберите команду Включить/Выключить Просмотр Шаг-за-Шагом.

 

Вставка точек прерывания

Следующее действие устанавливает или убирает точку прерывания на первом операторе в текущей строке алгоритма.
В меню Программа выберите команду Установить/Убрать точку прерывания.

 

Примечание:
В случае, когда в алгоритме есть точки прерывания, выполнение алгоритма будет остановлено в соответствующих точек. Для возобновления нормального выполнения алгоритма из точки прерывания выберите команду Запуск. Для возобновления выполнения алгоритма в пошаговом режиме из точки прерывания выберите команду Следующий Шаг.

 

Комментарии

В тексте алгоритма могут быть вставлены комментарии. Комментарии предназначены исключительно для пользователя - они облегчают понимание алгоритма. Текст комментарии вставляется между символами "{" и "}".
