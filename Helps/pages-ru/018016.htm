Условия Муравей

Муравей способен оценивать следующие условия:

  Край

 

Примечание:
Используйте отрицание Не для обозначения отрицания значения, получаемое при проверке условия.

Пример:
  Не Край

 

 

Типичные ситуации:

На вопрос: "Край"? для этих ситуаций Муравей ответит нет, то есть ложь.
На вопрос: "Не Край"? для тех же ситуаций Муравей ответит да, то есть истина.

На вопрос: "Край"? для этих ситуаций Муравей ответит да, то есть истина.
На вопрос: "Не Край"? для тех же ситуаций Муравей ответит нет, то есть ложь.
