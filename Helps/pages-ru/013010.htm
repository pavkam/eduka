Оператор цикла ПОВТОРИ n РАЗ

При составлении алгоритмов часто случается, что последовательность команд требуется повторить несколько раз подряд. В таких случаях для упрощения записи алгоритма можно использовать специальную команду - цикл Повтори n раз, который выполнит последовательность команд в теле цикла указанное число раз.

 

Цикл Повтори n раз описывается следующим образом:
 Повтори <n> Раз
   ...
  последовательность команд
   ...
 Конец Повтори

 

Где:
<n> - целое число, без знака, больше нуля.
