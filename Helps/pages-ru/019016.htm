Условия Пингвин

Пингвин способен оценивать следующие условия:

Будет Следовать Стена
Ячейка Содержит Букву
Карман Содержит Букву
Могу Вспомнить
Ячейка = Карман
Ячейка > Карман
Ячейка < Карман
Карман = Ячейка
Карман > Ячейка
Карман < Ячейка
Ячейка = Буква
Ячейка > Буква
Ячейка < Буква
Карман = Буква
Карман > Буква
Карман < Буква
Карман Содержит Прописную
Ячейка Содержит Прописную

 

Где:
Буква - Буква латинского алфавита.

 

Примечание:
Используйте отрицание Не для обозначения отрицания значения, получаемое при проверке условия.

Пример:

  Не Будет Следовать Стена
