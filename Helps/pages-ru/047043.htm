Сохранение файла

Для быстрого сохранения документа, используйте команду Сохранить на панели инструментов. Для сохранения копии документа под другим именем используйте ниже приведенные действия.

 

Сохранение копии документа

1. Из меню Файл выберите команду Сохранить Как.
2. В поле Имя файла введите новое имя файла.
3. Нажмите кнопку Сохранить.

 

Совет:
Для сохранения копии документа в другом каталоге, в поле Папка выберите диск или каталог, затем в поле, которое расположено ниже, выбрать (двойным щелчком) папку в котором хотите сохранить документ. Для сохранения документа в новом каталоге выберите команду Создать Новую Папку затем выберите её (двойным щелчком).

 

Экспорт в формате HTML текущего документа

1. Из меню Файл выберите команду Экспорт в HTML.
2. В поле Имя файла введите новое имя файла.
3. Нажмите кнопку Сохранить.

 

Примечание:
Eduka+ сохраняет файлы в специальном формате, что позволяет перевод содержимого на другой язык (переводятся ключевые слова алгоритмического языка). Поэтому не рекомендуется сохранение файла алгоритма, написанного на русском или румынском языке, в котором идентификаторы были написаны на соответственном языке. Для правильного перевода алгоритма используйте в написании идентификаторов только буквы латинского алфавита вне зависимости, на котором языке работаете.
