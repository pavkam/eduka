Создание задачи

В меню Инструменты выберите команду Редактор Задач.
Программа переходит в режим редактирования задач.

 

Следующее действие создает новую незаполненную задачу.

На панели инструментов, выберите команду Создать новую задачу.

Из меню:
Выберите команду Создать из меню Файл.

 

Созданная с помощью Eduka+ задача имеет следующее содержание:

1. Автор – имя (псевдоним) автора задачи.
2. Длительность – рекомендованное время, отведенное для решения задачи.
3. Уровень – уровеньсложности задачи.
4. Условия задачи – условия задачи с начальными и конечными условиями.
5. Ограничения задачи – ограничения и предписания задачи.
6. Начальное состояние и Конечное состояние поля.

 

Совет:
Если желаете обмениваться задачами с вашими коллегами и друзьями либо с другими пользователями Eduka+, заполните рубрики Условия задачи и Ограничения задачи на разных языках. Для этого выберите из списка Язык необходимый язык и затем заполните все незаполненные рубрики для соответственного языка.
Используйте команду Восстановить Сохранённое Поле чтобы указать одно из состоянии поля (Начальное или Конечное). Для этого предварительно в нормальном режиме программы (режим редактирования алгоритма) запомните состояние поля с помощью команды Сохранить Текущее Поле.
